# Code of Conduct

This is a kid-dad project — built together by a parent and a child for fun. We want everyone who interacts with the project (filing issues, sending pull requests, asking questions) to feel welcome.

## Expectations

When participating in this project's spaces (issues, pull requests, discussions):

- **Be kind.** Treat others with respect.
- **Keep it constructive.** Suggestions are great. Personal attacks aren't.
- **Remember a kid might read this.** Pick your language and topics accordingly.
- **Stay on topic.** This is a small space-shooter game; keep the discussion about that.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Discrimination based on age, gender, sexual orientation, disability, ethnicity, religion, or nationality.
- Sexual, violent, or otherwise NSFW content in any project space.
- Spam, off-topic promotion, or repeated derailing.

## Enforcement

The maintainer ([@marceld23](https://github.com/marceld23)) may edit or delete comments, close threads, or block contributors at their discretion. Decisions are final.

To report a problem, email **marcel.duetscher@gmail.com**.

## Scope

This code applies to all project spaces: GitHub issues, pull requests, discussions, the commit history, and any official communication channels.
